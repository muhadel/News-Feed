## Getting Started

To run this application locally, ensure that you have Node.js and Docker installed on your machine.

### Frontend

1. Install packages

```
pnpm install
```

2. Copy the environment file

```
cp .env.example .env.local
```

3. Run frontend

```
pnpm  dev
```
